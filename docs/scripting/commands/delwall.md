# `delwall()`

!!! warning
	This page may contain outdated information, incompatible with the current version of Hercules and its coding standards.

## Syntax

```c
delwall("<wallname>");
```

## Description

Deletes an invisible wall previously created by [`setwall()`](setwall.md); identified by the wall's name.
